TaskTracker alkalmazás főbb technológiái

BE: Java 21, Spring Boot 3.5.7

FE: Vue.js 3.3.4, Vite 4.4.9, Axios 1.5.0

DB: MySQL

Az alkalmazás localban működik, DB-t XAMPP-al hostolni és app.props-ban szereplő user-t és db-t létrehozni futtatáshoz
BE: IDE-ben backend megnyitása projektként és futtatni
FE: frontend mappa megnyitása után terminalban (VSCODE) npm install majd npm run dev és elindul a Vite szerver, vite url-t megnyitva megjelen a kezelőfelület
